# Dialob Review Components for Material UI

## Application Setup

### Dependencies

Add dependency

```
yarn add @resys/dialob-review-material
```

Following peer dependencies need to be installed:

```json
{
  "@material-ui/core": "^4.11.*",
  "@material-ui/icons": "^4.9.*",
  "@material-ui/lab": "^4.0.*",
  "@material-ui/pickers": "^3.2.*",
  "@date-io/moment": "^1.3.*",
  "moment": "^2.24.*",
  "react-intl": "5.3.*",
  "react-markdown": "^4.3.*"
}
```
